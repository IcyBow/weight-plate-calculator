How “most efficient” is chosen

For each achievable per-side weight, the app chooses the combination with the fewest plates per side. If there is a tie, it chooses the one using the fewest different plate types.

All calculations are exact using grams internally (no floating point drift). Balanced means the same plates on both sides.

Made for offline/local use. No network requests. Data stays in your browser.
