# weight-plate-calculator
This has been something I wanted to do for quite some time. I was tired of always redoing the same calculations and not knowing easily what the next increment in weight would be for my homegym.

So with the help of AI, this very simple webapp was generated. It enables you to create a printable list of all the weight combinations with your exact setup.
I have no plans of upgrading it more since it fits exactly what I want.
